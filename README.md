# Tareas
Aplicación para gestionar tareas

- Crear tarea.
- Eliminar tarea.
- Cambiar estado de tarea.


## Compatibilidad

* ANGULAR: version 11.0.2.
* NODE: version :12.18.3

## Dependencias necesarias:

Descargar los módulos necesarios: `npm install`

## Run project

Run `ng serve -0` to exec the project.
